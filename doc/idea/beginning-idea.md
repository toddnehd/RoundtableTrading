# 멀티 에이전틱 주식 트레이딩 서비스

## 1. 개요
- 다양한 분야의 AI 에이전트가 시장, 종목, 기업 가치, 차트, 기술적 지표 등을 분석.
- 각 분야에 특화된 에이전트들은 서로 시장, 종목에 대해 논의하고 주가를 예측 및 사용자에게 추천.
- 주식 시세를 확인하고 정해진 알고리즘 또는 분석한 정보에 따라 매수, 목표가, 손절가를 실시간으로 예측.
- 자동매매 모드에서는 증권사의 open api 에 연결된 mcp 를 통해 직접 에이전트가 투자를 판단하고 매매를 진행.
- 현재는 한국의 KOSPI, KOSDAQ 시장의 주식 종목을 대상으로만 위 부분들을 처리. 추후 투자 대상 및 시장을 늘려갈 예정.

## 2. 멀티 에이전트
- 기업 가치 즉, 기업의 펀터멘털, 재무재표, 벨류에이션, 미래 성장성, 기업의 뉴스 등을 분석하고 이에 맞는 의견을 제시하는 에이전트.
- 테마, 업종, 섹터 별 특화 에이전트 (분류에 따라 그 숫자가 많아질 수 있음).
- 실시간 뉴스 및 환율, 지표등을 통해 국내, 해외의 시장 변화를 예측하고 단일 종목 보다는 좀 더 큰 흐름에서 판단을 할 수 있는 에이전트
- 종목의 모멘텀, 거래량, 거래대금, 수급(개인, 기관, 외국인, 연기금 등), 차트, 기술적 지표에 따라 주가를 예측하는 에이전트
- (분야별 에이전트는 아이디어에 따라 더 추가될 수 있음.)

## 3. 에이전트 간의 토론
- 단순히 하나의 에이전트의 결정에 따라 주가 예측 및 트레이딩을 하는게 아닌 각 분야의 에이전트가
마치 원탁에 앉아 대화하듯 토론하고 합의에 이른 종목을 대상으로 트레이딩을 진행.
- 추천 종목, 매수 가격, 목표 가격, 손절 가격, 진입 시점 등을 도출할 수 있어야함.
- 주문 뿐 아니라 현재 보유중인 포트폴리오를 분석하고 위험 감지, 목표가 재설정 등을 통해 재분배가 가능해야함.

## 4. 도구
- 증권사 open api 를 연동하여 실제 주문을 낼 수 있고, 잔고, 시세 등을 조회할 수 있는 mcp 서버.
- pykrx 와 같이 증권 웹사이트에서 주식 정보를 스크래핑 할 수 있는 라이브러리
- 그 외 시세 데이터를 통해 기술적 지표를 생성 할 수 있는 파이썬 도구들
